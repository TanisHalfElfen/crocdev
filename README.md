# Crocadile Housing Development In Scissons Rd - Sawyer Way - Landover Crescent Neighbourhood
Scissons–Sawyer-Landover Area Development Review (Crocodile Private Proposal)

This repository documents publicly available information relating to the proposed redevelopment at 10–36 Crocodile Private in Kanata South.

The purpose is to organize information, summarize proposal changes, and provide a reference point for residents seeking factual details.

Crocodile Private Development Proposal
Structured Summary (February 2026)

1. Background

  In 2017, the Ontario Municipal Board approved a subdivision plan at 27–35 Scissons Road for:
  15 single-family detached homes
  Private street (Crocodile Private)
  
  Tree retention block along the southerly lot line
  Built form consistent with surrounding two-storey homes
  Public consultation occurred in 2015–2016.

2. Current Proposal (February 10, 2026 Notification Letter)

  According to a neighbour notification letter dated February 10, 2026 from GEMTEC Consulting Engineers and 
  Scientists (on behalf of Pegasus Development Corporation), the owner of 10–36 Crocodile Private intends to submit 
  Planning Act applications to redevelop the lands.
  The proposal includes:
  
  30 three-storey single-detached homes
  Crescent-shaped private roadway (Crocodile Private)
  Two internal entrances from Scissons Road
  Approximately 1042 sq ft footprint per dwelling
  14 homes fronting directly onto the private roadway
  16 homes accessed via short extensions of the roadway

  Required applications:
  
  Zoning By-law Amendment
  Plan of Condominium
  Site Plan Control
  Part Lot Control

  The letter notes that pre-consultation discussions with City of Ottawa staff occurred in May 2024.

3. Change in Unit Yield

  The proposal represents an increase from the previously approved 15 units to 30 units.
  This constitutes a 100% increase in dwelling count compared to the 2017 approval.

4. Neighbourhood Context

  The Scissons / Sawyer Way / Landover Crescent area functions as an internal residential road network served by a single outlet connection to Stonehaven Drive.
  Based on visual count of existing dwellings in the immediate area:
  Approximately 80–90 existing homes currently share this access route.
  If 30 additional units are approved, the total number of dwellings served by the same outlet would be approximately 110–120 homes.

  (This is an approximate estimate based on publicly visible housing counts.)

5. Process Status

  At the time of the February 10, 2026 letter:
  Formal applications had not yet been circulated by the City.
  Official public notification and comment periods will follow once the applications are submitted and processed.
  Residents will have the opportunity to provide input through the City’s formal review process.

6. Purpose of This Repository

  This repository is intended to:
  Archive publicly available documents
  Summarize proposal changes
  Track timelines
  Provide a central reference point for residents
